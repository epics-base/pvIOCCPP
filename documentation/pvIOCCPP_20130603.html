Abstract

Thus project is obsolete. See status for details.

It was one part of the set of related products in the EPICS V4 control system programming environment:

pvData
pvData (Process Variable Data) defines and implements an efficent way to store, access, and communicate memory resident structured data
pvAccess
pvAccess is a software library for high speed controls network communications, optimized for pvData
pvIOC
pvIOC is a software framework for building network accessable "smart" real time databases, suitable for interfacing devices in a distributed control system, that can exchange pvData over pvAccess.
pvService
A middle layer for implementing efficient data services.

Each of these products has a Java and a C++ implementation.

The products are all part of the V4 implementation of Experimental Physics and Industrial Control System (EPICS).

Status of this Document

This is the 03-Jul-2013 version of the C++ implementation of pvIOC. This project is obsolete. All of it's functionality has been moved elsewhere.

The v3Channel support has been moved to pvaSrv. pvaSrv uses channelProviderLocal and channelBase. Equivalent support is now provided by pvAccessCPP. When pvaSrv is changed to use the pvAccess support this project will become empty.

The ezchannelRPC.h suppport is now implemented by pvAccessCPP. The service.h and pvServiceProvider. h support is now provided by pvAccessCPP. No other code is currently provided by this project.
